Projects and Portfolio

My main works where I turn theoretical knowledge into practical applications, showcasing my technical skills and creativity:

GNSS-Free Positioning with Terrain Features and Visual Data

A navigation project developed to ensure autonomous navigation for UAVs in areas of critical signal loss. The system aims for precise positioning using terrain recognition, map matching (similar to SLAM), and sensor fusion techniques integrated with Digital Elevation Models (DEM/DSM). (November 2025 - Ongoing)

GPS Accuracy Analysis using LiDAR Data Models

A scientific study evaluating the vertical and horizontal accuracies of data obtained from various GPS/GNSS devices, using high-resolution Elevation Models (DEM/DSM) derived from LiDAR as a reference. This project earned me a Finalist title in the Esri Turkey Young Academics Competition. (2025)

Raspberry Pi Drone System with Image Processing Capability

Designed and developed using a Raspberry Pi and an F-450 chassis to create a low-cost, customizable drone platform. The focus was on developing efficient image capture, telemetry, and data processing software solutions. (September 2023 - November 2024)
